臺灣彰化地方法院刑事判決　　　　　　　95年度訴字第1102號
公　訴　人　臺灣彰化地方法院檢察署檢察官
被　　　告　甲○○
　　　　　　丁○○
上　一　人
選任辯護人　張崇哲律師
　　　　　　林志銘律師
上列被告等因廢棄物清理法案件，經檢察官提起公訴（95年度偵
字第3819號），本院判決如下：
    主  文
甲○○共同竊盜，處有期徒刑拾月；又共同連續未依廢棄物清理
法規定領有廢棄物清除、處理許可文件，從事廢棄物處理，處有
期徒刑壹年肆月，未扣案之挖土機壹輛沒收。應執行有期徒刑貳
年，未扣案之挖土機壹輛沒收。
丁○○幫助竊盜，處有期徒刑陸月，如易科罰金以銀元佰元即
新臺幣玖佰元折算壹日。
    犯 罪 事 實
一、緣甲○○知悉真實姓名年籍不詳，綽號「噴火」之成年男子
    欲挖採他人土地之土石變賣，又得知坐落於彰化縣溪州鄉○
    ○○段490之347地號國有耕地（下稱系爭土地）之承租人乙
    ○○因身體健康狀況不佳，無法繼續耕作，而欲將其對系爭
    土地之耕作權讓渡予他人，竟計畫受讓乙○○之耕作權後，
    再以不詳代價（至少近百萬元）將系爭土地交予綽號「噴火
    」之成年男子竊取土石以牟利，而丁○○亦明知上情，仍基
    於幫助竊盜之犯意，於民國（下同）91年5月14日，應甲○
    ○之邀約，陪同甲○○前往乙○○位於彰化縣溪州鄉○○村
    ○○路11之6號住處，與不知情之乙○○（業經檢察官為不
    起訴處分確定）洽談讓售系爭土地耕作權事宜，雙方議定以
    新臺幣76萬元讓售系爭土地耕作權後，丁○○旋又陪同甲○
    ○前往找尋不知情之丙○○，以5千元之代價邀約其擔任系
    爭土地耕作權之名義上受讓人，丙○○因身染毒癮惡習，需
    錢孔急，而未加詢問源由即應允之（丙○○部分亦經檢察官
    為不起訴處分確定），並與甲○○、丁○○一同前至乙○○
    所指定，設於彰化縣溪州鄉庄南巷1號之7「黃雨田代書事務
    所」，以其名義與乙○○簽署由不知情之黃百堅代為擬定之
    「承租公有土地耕作權讓渡證書」，復由甲○○當場交付76
    萬元予乙○○點收後，雙方各自離去；而甲○○則在回程途
    中，向丁○○借貸5千元予丙○○以為其充當系爭土地耕作
    權名義受讓人之報酬。甲○○於取得系爭土地耕作權後，其
    與綽號「噴火」之成年男子均明知系爭土地係屬財政部國有
    財產局所有，未經財政部國有財產局同意，不得擅自挖採其
    上土石，竟仍意圖為自己不法之所有，並基於共同犯意之聯
    絡，於91年年底，由綽號「噴火」之成年男子僱工駕駛挖土
    機及砂石車至系爭土地挖採土石（尚無證據足認挖採土石時
    ，有具共同犯意聯絡之三人以上同時在場），共計於系爭土
    地及與之附連圍繞之同地段第589地號土地上竊得土石21968
    .82 立方公尺，甲○○則因而獲得至少近百萬元之代價。後
    於94年年中，一真實姓名年籍不詳之成年男子見系爭土地上
    留有一大坑洞，經探詢後得知系爭土地係由甲○○管領使用
    ，遂向甲○○表示欲以每車1千元之代價於該處傾倒一般事
    業廢棄物，甲○○明知未依廢棄物清理法規定領有廢棄物清
    除、處理許可文件，不得從事廢棄物清除、處理之工作，竟
    為獲取高額之代價而應允之，並與該真實姓名年籍不詳之成
    年男子共同基於違法清除、處理一般事業廢棄物之概括犯意
    聯絡，先後多次由該真實姓名年籍不詳之成年男子自不詳處
    所載運含有造紙工廠生產後剩餘之散漿渣及摻雜塑膠袋、磚
    瓦、廢木板、帆布、廢土之營建混合物等一般事業廢棄物至
    系爭土地傾倒，甲○○則負責僱工或自行駕駛其所有之挖土
    機（未扣案）將所傾倒之前開廢棄物予以掩埋、整地，期間
    並因前開廢棄物起火悶燒，甲○○尚自行駕駛挖土機挖掘引
    水道，導引系爭土地附近之水源即八號深井之水流至系爭土
    地，並以其挖土機翻攪該等廢棄物滅火。嗣於95年2月17日
    上午10時30分許，經警發覺系爭土地遭人傾倒廢棄物，遂會
    同彰化縣政府環境保護局稽查隊人員到場稽查，始循線查悉
    上情。
二、案經彰化縣警察局北斗分局報請臺灣彰化地方法院檢察署檢
    察官偵查起訴。
        理 由
一、按被告以外之人於審判外之陳述，除法律有規定者外，不得
    作為證據；又被告以外之人於審判外之陳述，雖不符刑事訴
    訟法第159條之1至第159條之4規定，而經當事人於審判程序
    同意作為證據，法院審酌該言詞陳述或書面陳述作成時之情
    況，認為適當者，亦得為證據。當事人、代理人或辯護人於
    法院調查證據時，知有第159條第1項不得為證據之情形，而
    未於言詞辯論終結前聲明異議者，視為有前項之同意，刑事
    訴訟法第159條第1項、第159條之5定有明文。立法意旨在於
    傳聞證據未經當事人之反對詰問予以核實，原則上先予排除
    。惟若當事人已放棄反對詰問權，於審判程序中表明同意該
    等傳聞證據可作為證據；或於言詞辯論終結前未聲明異議，
    基於尊重當事人對傳聞證據之處分權，及證據資料愈豐富，
    愈有助於真實發見之理念，且強化言詞辯論主義，使訴訟程
    序得以順暢進行，上開傳聞證據亦均具有證據能力。查本案
    除被告丁○○及其選任辯護人以證人丙○○、乙○○於警詢
    中之證述係傳聞證據，而認不具證據能力外，其餘後述認定
    事實所憑之供述證據及非供述證據，公訴人、被告2人及選
    任辯護人均不爭執其證據能力，且經本院於審判程序時提示
    並告以要旨，公訴人、被告2人及選任辯護人均知悉上述供
    述證據屬傳聞證據，而亦未於言詞辯論終結前聲明異議，本
    院復衡以該等證據作成時之客觀條件及環境，認為適當，是
    可認後述引用該等證據之證據能力均無疑義。至被告丁○○
    及其選任辯護人爭執證人丙○○、乙○○於警詢中證述之證
    據能力部分，按被告以外之人於檢察事務官、司法警察官或
    司法警察調查中所為之陳述，與審判中不符時，其先前之陳
    述具有較可信之特別情況，且為證明犯罪事實存否所必要者
    ，得為證據，刑事訴訟法第159條之2定有明文，足見被告以
    外之人於司法警察（官）調查中所為之陳述，依刑事訴訟法
    第159條第1項規定，係屬傳聞證據，原則上無證據能力，惟
    如該陳述與審判中不符時，其先前之陳述具有較可信之特別
    情況，且為證明犯罪事實存否所必要者，依同法第159條之2
    規定（即刑事訴訟法第159條第1項所指之「除法律有規定者
    外」），始例外認為有證據能力；如該陳述與審判中相符時
    ，因該陳述並不符合刑事訴訟法第159條之2有關傳聞例外之
    規定，故不得作為認定本案犯罪事實有無之證據，此時，當
    以其於審判中之陳述作為證據。另所謂「前後陳述不符」之
    要件，應就前後階段之陳述進行整體判斷，以決定其間是否
    具有實質性差異，惟無須針對全部陳述作比較，陳述之一部
    分有不符，亦屬之。查證人丙○○、乙○○於警詢中之證述
    ，因屬審判外之陳述，為傳聞證據，又其等整體陳述核與於
    本院審判中之證述大致相符，是其2人於警詢中之陳述並不
    符合刑事訴訟法第159條之2有關傳聞例外之規定，被告丁○
    ○及其選任辯護人既已於本院準備程序時爭執其2人警詢中
    證述之證據能力，依前開說明，自不得作為認定被告丁○○
    犯罪事實所憑之證據，而應以證人丙○○、乙○○於審判中
    之證述作為證據，均合先敘明。
二、訊據被告丁○○固坦承有於91年5月14日陪同同案被告甲○
    ○前往乙○○住處洽談讓售系爭土地耕作權事宜，並於同日
    一同至「黃雨田代書事務所」，由丙○○代同案被告甲○○
    與乙○○簽訂「承租公有土地耕作權讓渡證書」，事後且借
    貸5千元予同案被告甲○○交給丙○○等情不諱，惟矢口否
    認有何共同竊盜及違反廢棄物清理法第46條之犯行，辯稱：
    伊確實曾懷疑甲○○為何要請丙○○代為簽立讓渡書，但伊
    是簽約後方知道甲○○受讓系爭土地係欲供人竊取系爭土地
    之土石云云。另被告甲○○則固坦承有上開竊盜及違反廢棄
    物清理法第46條之犯行，然亦為同案被告丁○○辯稱：丁○
    ○陪同伊處理系爭土地耕作權受讓事宜時，並不知道伊受讓
    系爭土地之目的，丁○○雖有詢問，但伊係於簽約完後方告
    知其云云。惟查：
（一）上開被告甲○○與丁○○於91年5月14日，一同前往與證
      人乙○○洽談系爭土地耕作權讓售事宜，復於同日一同至
      「黃雨田代書事務所」，由被告甲○○向被告丁○○借貸
      5千元予證人丙○○，由證人丙○○以其名義與證人乙○
      ○簽訂系爭土地之「承租公有土地耕作權讓渡證書」；嗣
      於同年年底，被告甲○○再以至少近百萬元之代價，將系
      爭土地交予綽號「噴火」之成年男子竊取其上土石牟利，
      共計於系爭土地及與之附連圍繞之同地段第589地號土地
      上竊得21968.82立方公尺之土石；後於94年年中，被告甲
      ○○另以每車1千元之代價，同意一真實姓名、年籍不詳
      之成年男子先後多次自不詳處所載運含有造紙工廠生產後
      剩餘之散漿渣及摻雜塑膠袋、磚瓦、廢木板、帆布、廢土
      之營建混合物等一般事業廢棄物至系爭土地傾倒，被告甲
      ○○並負責僱工或自行駕駛其所有之挖土機將所傾倒之前
      開廢棄物予以掩埋、整地等情，除據被告2人迭於本院準
      備程序與審理時均坦承不諱，並據證人乙○○、丙○○分
      別於偵訊中及本院審理時；證人即系爭土地附近公有地之
      承租人宋添喜、余福富、宋鄭萍、宋清興分別於警、偵訊
      中；證人即彰化縣環境保護局稽查人員林新育、曾國偉、
      張志豪於偵查中；證人即國有財產局人員蕭乙中於偵查中
      ，均證述綦詳，復有彰化縣環境保護局稽查隊環境稽查工
      作紀錄、承租公有土地耕作權讓渡證書、系爭土地附近之
      水源處八號深井之收取抽水馬達電費資料、彰化縣政府95
      年6月2日府工水字第095014211號函、彰化縣北斗地政事
      務所95年6月2日北第二字第0950002679號函暨所檢附之鑑
      定圖、地籍參考圖各1份及現場照片20幀在卷可資佐證，
      此部分事實應堪認定。
（二）被告2人雖於本院審理均以前詞置辯，然證人丙○○已於
      本院審理明白結證稱：被告2人係於系爭土地簽約當日，
      先一同來找伊，以5千元之代價，要伊以伊名義受讓系爭
      土地，伊因染上毒癮，亟需用錢便答應其2人，隨後伊等3
      人旋即一同至代書那簽約，簽約完於回程途中，甲○○向
      丁○○說沒有5千元，要其先借用，丁○○就直接拿5千元
      予伊，整個過程丁○○都有在場，但伊已無法記憶是何人
      負責與伊談論此事；伊與丁○○是小學同學，與甲○○僅
      認識幾年，與丁○○較為熟識，伊所言均是事實，絕無刻
      意誣陷任何人之情等語；另證人乙○○亦於本院審理時結
      證稱：伊與被告2人原均不認識，伊因身體狀況不佳，無
      法繼續耕作，而曾告知他人欲轉讓系爭土地耕作權之訊息
      ，後來甲○○即先至伊住處表示欲受讓系爭土地耕作權，
      伊開價76萬元，甲○○認為價格太高而未成交，第2次甲
      ○○與丁○○一同至伊住處，再次與伊洽談此事，甲○○
      剛開始有殺價，但伊仍堅持，最後即以76萬元成交，在洽
      談過程中，主要是甲○○負責與伊談，丁○○只是在一旁
      幫忙說而已，談成後同日，伊等即相約至代書那裡簽約，
      伊是自行到代書處，而甲○○、丁○○則是與丙○○一同
      至代書處，由丙○○與伊簽約，簽約後由甲○○交付予伊
      現金76萬元，雙方即各自離去；簽約當日從在伊住處洽談
      至到代書那裡簽約，丁○○均有在場，而丙○○則是在代
      書那裡才出現，其並未與被告2人一同至伊住處洽談等語
      ，是以被告丁○○參與系爭土地締約過程之情形，參諸被
      告2人熟稔之程度及被告2人當時均係砂石車職業駕駛，對
      此事應有之敏感度，被告丁○○豈會對被告甲○○受讓系
      爭土地耕作權之源由毫無所悉？且被告丁○○於本院審理
      時亦自承確有懷疑被告甲○○為何要找證人丙○○代為簽
      約？而被告甲○○亦於本院審理時證述：伊覺得與丁○○
      是朋友，所以向其說伊要盜採砂石也沒關係等語，益徵被
      告丁○○不可能於事前絲毫不知情；再苟其確係事後方知
      悉，為何於警、偵訊中均未據實供承，而先於警詢中否認
      知悉系爭土地簽約之事，後於偵訊中又辯稱其2人係陪同
      證人丙○○前往簽約云云，顯見被告丁○○情虛之情，況
      其於本院準備程序時已自承：簽約時確已知悉被告甲○○
      受讓系爭土地耕作權係欲供盜採砂石所用等語，堪認被告
      2人所辯被告丁○○係事後方知情乙節，純係事後卸責之
      詞，不足採信。是以被告2人之交情，及其等當時之職業
      ，兼衡一般人於陪同他人前往處理事情之前，即會詳加詢
      問源由之常情，堪認被告丁○○於陪同被告甲○○一同前
      往與乙○○洽談系爭土地受讓事宜時，即應已知悉被告甲

（三）按現行刑法關於正犯、從犯之區別，係以其主觀之犯意及
      客觀之犯行為標準，凡以自己犯罪之意思而參與犯罪，無
      論其所參與者是否犯罪構成要件之行為，皆為正犯，其以
      幫助他人犯罪之意思而參與犯罪，其所參與者，苟係犯罪
      構成要件之行為，亦為正犯，必以幫助他人犯罪之意思而
      參與犯罪，其所參與者又為犯罪構成要件以外之行為，始
      為從犯（最高法院25年上字第2253號判例意旨參照）；復
      按以自己共同犯罪之意思，事先同謀，而由其中一部分人
      實施犯罪之行為者，均為共同正犯，司法院大法官會議釋
      字第一○九號解釋可資參照，而此即所謂之「共謀共同正
      犯」；又共同實施犯罪行為之人，在合同意思範圍以內，
      各自分擔犯罪行為之一部，相互利用他人之行為，以達成
      其犯罪目的者，即應對全部所發生之結果，共同負責，是
      共同正犯在犯意聯絡範圍內之行為，應同負全部責任（最
      高法院28年上字第3110號判例意旨參照）。查被告甲○○
      係欲以近百萬元之代價，將系爭土地交予綽號「噴火」之
      成年男子竊取其上土石，藉以獲取暴利；嗣又以每車次1
      千元之代價，由不詳之成年男子載運一般事業廢棄物至系
      爭土地傾倒，並由其負責掩埋、整地，均顯係以自己犯罪
      之意思而參與犯罪，並分別與綽號「噴火」及上開不詳姓
      名之成年男子基於共同犯意之聯絡，於合同意思範圍內，
      相互利用他人之行為，以達成其等之犯罪目的，依上開說
      明，被告甲○○自應分別與綽號「噴火」之成年男子及上
      開不詳姓名之成年男子共負共同正犯之責。另被告丁○○
      雖明知同案被告甲○○受讓系爭土地耕作權係欲供他人竊
      取其上土石所用，而仍陪同其前往洽談及簽訂系爭土地耕
      作權讓售事宜，甚陪同被告甲○○前往找尋丙○○代為簽
      立讓渡書，並借貸5千元予被告甲○○以供其給付丙○○
      充當名義上受讓人之報酬，然其並未參與任何竊取土石之

      與被告甲○○事前同謀，而推由被告甲○○負責實施竊盜
      犯行之情事，亦難認被告丁○○與被告甲○○有何共同竊

（6）另按關於易科罰金及定應執行之部分，參酌最高法院95年

（二）核被告甲○○所為，係分別犯刑法第320條第1項之竊盜罪

（三）又被告2人前揭行為後，刑法關於共犯、連續犯及罰金刑
      等規定已有修正，依前開新舊法適用原則，經本院綜合其
      等全部罪刑之結果而分別為如附表1之1、1之2及附表2所
      示之比較後，認本件依刑法第2條第1項前段規定，就被告
      2人以下分別關於共犯、連續犯及罰金刑等規定，均應依
      舊刑法之規定。是被告甲○○與綽號「噴火」之成年男子
      就竊盜犯行間；及被告甲○○與上開真實姓名、年籍不詳
      之成年男子及其所僱用駕駛挖土機將所傾倒廢棄物予以掩
      埋、整地之人員就未依規定領有許可文件，從事廢棄物處
      理之犯行間，分別有犯意聯絡及行為分擔，均為共同正犯
      。被告甲○○利用不知情之丙○○與系爭土地之原承租人
      乙○○簽訂耕作權受讓契約後，將系爭土地提供予綽號「
      噴火」之人竊取土石，為間接正犯。又按連續犯之成立，
      除主觀上須基於一個概括之犯意外，客觀上須先後數行為
      ，逐次實施而具連續性，侵害數個同性質之法益，其每一
      前行為與次行為，依一般社會健全觀念，在時間差距上，
      可以分開，在刑法評價上，各具獨立性，每次行為皆可獨
      立成罪，構成同一之罪名，始足當之；如數行為於同時同
      地或密切接近之時地實施，侵害同一之法益，各行為之獨
      立性極為薄弱，依一般社會健全觀念，在時間差距上，難
      以強行分開，在刑法評價上，以視為數個舉動之接續施行
      ，合為包括之一行為予以評價，較為合理，則屬接續犯，
      而為包括之一罪（最高法院86年臺上字第3295號判例意旨
      參照）。查被告甲○○係基於與綽號「噴火」之成年男子
      同一之砂石買賣契約，而將系爭土地交予其挖採，雖持續
      一段期間，然被告甲○○係基於同一主觀犯意，完成同一
      犯罪目的，雖外觀上有數舉動，但各舉動係於密切接近之
      時間實施，侵害同一法益，依上開判例意旨，應論以一接
      續行為較符法理；另被告甲○○先後多次以每車次1千元
      之代價，允許他人於系爭土地上傾倒廢棄物後，由其負責
      予以掩埋整地之行為，其每次行為前後均相隔數日以上，
      並非密切接近之實施，在時間差距上，可以分開，在刑法
      評價上，各具獨立性，每次行為皆可獨立成罪，然其每次
      犯行之時間仍緊接，方法相同，所犯又係構成要件相同之
      罪，主觀上應係基於概括犯意反覆為之無訛，是依前開判
      例意旨，被告先後多次未依規定領有許可文件，從事廢棄
      物處理之犯行，應論以連續犯，應依修正前刑法第56條規
      定以一罪論，並加重其刑。再按刑法第55條所謂之牽連犯
      ，必須2個以上之行為有方法與結果之關係者，始足構成
      ，亦即必須以犯1罪之方法行為犯他罪，或以犯一罪之結
      果行為犯他罪，方有牽連關係之可言（最高法院73年臺上
      字第5446號判例意旨參照）。查被告甲○○於91年間受讓
      系爭土地承租權時，原僅係欲將系爭土地提供予綽號「噴
      火」之成年男子竊取砂石所用，嗣於94年間，始因上開真

      經比較如附表4後，因被告甲○○行為後即裁判時之新法
      亦未較為有利，依刑法第2條第1項前段規定，亦應適用舊
      法第51條規定定其應執行刑。末查被告甲○○用以處理前
      開一般事業廢棄物所用之挖土機1輛，係被告甲○○所有
      ，且已供其違法從事廢棄物處理犯罪所用之物，業據其自
      承在卷，雖未扣案，然尚無證據證明業已滅失，兼衡其違
      法從事廢棄物處理所獲之利益及該輛挖土機之價值又非顯
      不相當，爰依修正前刑法第38條第1項第2款併予宣告沒收
      。
四、公訴意旨另以：被告丁○○明知未經主管機關許可，不得任
    意堆置事業廢棄物，竟與被告甲○○共同基於犯意之聯絡，
    於94年間，由其2人自行，或委任之不詳人士，以不詳方法
    載運造紙工廠生產後剩餘之散漿渣及營建混合物（參雜塑膠
    袋及廢土）等事業廢棄物至系爭土地上之坑洞傾倒、堆置。
    因認被告丁○○另與被告甲○○共同涉犯廢棄物清理法第46
    條第1項第4款之罪嫌。惟按犯罪事實應依證據認定之，無證
    據不得認定其犯罪事實，刑事訴訟法第154條第2項定有明文
    。次按認定不利於被告之事實，須依積極證據，苟積極證據
    不足為不利於被告事實之認定時，即應為有利於被告之認定
    ，更不必有何有利之證據，縱認被告抗辯或反證係屬虛偽，
    仍不能以此資為積極證據應予採信之理由；又按事實之認定
    ，應憑證據，如未能發現相當證據，或證據不足以證明，自
    不能以推測或擬制之方法，以為裁判基礎（最高法院30年上
    字第816號、30年上字第482號、40年臺上字第86號判例參照
    ）。再按認定犯罪事實所憑之證據，雖不以直接證據為限，
    間接證據亦包括在內，然無論直接或間接證據，其為訴訟上
    之證明，須於通常一般之人均不致於有所懷疑，而得確信其
    為真實之程度者，始得據為有罪之認定，倘其證明尚未達到
    此一程度，而有合理之懷疑存在時，即無從為有罪之認定（
    最高法院76年臺上字第4986號判例參照）。另按刑法上之「
    共同正犯」係指以自己共同犯罪之意思，事先同謀，推由其
    中一部分人實施犯罪之行為，或以自己犯罪之意思而參與犯
    罪，或已參與犯罪構成要件之行為者謂之（司法院大法官解
    釋第109號解釋、最高法院25上字第2253號判例意旨參照）
    。本件公訴人認被告丁○○涉有此部分犯行，無非係以證人
    乙○○、丙○○、林新育、曾國偉、張志豪分別於偵訊中及
    本院審理時之證述；彰化縣環境保護局稽查隊環境稽查工作
    紀錄及現場照片20 幀為其主要論據；然訊據被告丁○○則
    堅詞否認有何違反廢棄物清理法之犯行，辯稱：伊根本未自
    行或委任他人載運任何廢棄物至系爭土地傾倒、堆置，伊係
    事後方聽他人說系爭土地被甲○○傾倒廢棄物等語。經查：
（一）本件系爭土地雖確遭人傾倒、堆置一般事業廢棄物，此業
      據證人林新育、曾國偉、張志豪於偵訊中證述明確，並有
      彰化縣環境保護局稽查隊環境稽查工作紀錄1份及現場照
      片20幀可按，然依證人宋添喜、余福富、宋鄭萍、宋清興
      分別於警、偵訊中之證述均僅提及曾見被告甲○○於系爭
      土地指揮他人或自行駕駛挖土機整地，及曾自行駕駛挖土
      機挖掘引水道企圖熄滅廢棄物因悶燒所產生之火苗等情，
      而均未證述被告丁○○曾至系爭土地清除、處理前開一般
      事業廢棄物之情，同案被告甲○○亦於本院審理時證述：
      丁○○並不知道、亦未參與本件任何載運廢棄物至系爭土
      地上傾倒之行為，更未因而分得任何利益等語，堪認被告
      丁○○所辯並非無稽，復無其他證據足認被告丁○○曾參
      與任何違反廢棄物清理法第46條第1項之構成要件行為，
      亦無證據足認其有與被告甲○○事前同謀，而推由被告甲
      ○○負責實施犯行之情事，或與被告甲○○有共同違反廢
      棄物清理法之犯意聯絡，依上開說明，自難令與被告甲○
      ○共負違反廢棄物清理法第46條第1項第4款之罪之共同正
      犯罪責。
（二）又證人乙○○、丙○○於偵訊中及本院審理時之證述，雖
      可據以認定被告丁○○明知被告甲○○受讓系爭土地係欲
      供人竊取土石之用，而仍基於幫助之故意，陪同其前往洽
      談及簽約，對之施以助力等情，業如前述，然按幫助犯係
      從屬於正犯而成立，並無獨立性，故幫助犯須對正犯之犯
      罪事實，具有共同認識而加以助力，始能成立，其所應負
      責任，亦以與正犯有同一認識之事實為限，若正犯所犯之
      事實，超過其共同認識之範圍時，則幫助者事前既不知情
      ，自不負責（最高法院75年度臺上字第1509號判決意旨參
      照），查被告甲○○於本院審理時已明白供述：伊受讓系
      爭土地之初原僅係欲供綽號「噴火」之成年男子竊取砂石
      所用，嗣於94年年中，一真實姓名、年籍不詳之成年男子
      來找伊表示欲以每車次1千元之代價，載運廢棄物至系爭
      土地傾倒，伊方與其開始違法從事廢棄物之清除、處理等
      語，是正犯即被告甲○○於簽約之初既無違法處理廢棄物
      之犯意，而係於事隔2年多後，方另行起意為之，則從犯
      即被告丁○○於幫助之初是否能對此部分事實有所預料或
      認識，確有可疑，況被告丁○○、甲○○於本院準備程序
      時均係供承：丁○○於簽約時僅知甲○○受讓系爭土地欲
      供盜採砂石所用而已等語，益徵被告甲○○事後於系爭土
      地違法處理廢棄物之行為，確已超出被告丁○○於幫助時
      主觀上所認識之範圍，依上開判決意旨，亦難令其負幫助
      違法處理廢棄物之罪責。
（三）從而，公訴人所舉之證據尚不能使本院獲致被告丁○○確
      有違反廢棄物清理法第46條第1項第4款之罪之確切心證，
      自不能以推測或擬制之方法即認被告丁○○確有公訴人所
      指之此部分犯行。此外，本院在得依或應依職權調查證據
      之範圍內，復查無其他積極證據足認被告丁○○確有何違
      反廢棄物清理第46條之犯行，揆諸前揭說明，不能證明被
      告丁○○此部分犯罪，原應為無罪之諭知，惟公訴人認為
      此部分與前揭經起訴認定有罪部分有修正前牽連犯之裁判
      上一罪關係，爰不另為無罪判決之諭知。
五、另被告甲○○就同案被告丁○○是否有幫助竊盜犯行一節，
    於本院審理中具結為證述，然其所證情節竟與其於本院準備
    程序時所供相互矛盾，且此節就被告丁○○是否為本件竊盜
    犯行之幫助犯而言，核屬案情有重要關係之事項，是被告甲
    ○○是否另涉犯刑法第168條之偽證罪嫌，應由檢察官另行
    偵辦，特予敘明。
據上論斷，依刑事訴訟法第299條第1項前段，廢棄物清理法第46
條第1項第4款，刑法第2條第1項前段、第11條前段、第28條（修
正前）、第30條第1項前段、第2項、第56條（修正前）、第320
條第1項、第41條第1項（修正前）、第38條第1項第2款（修正前
）、第51條第5款（修正前），罰金罰鍰提標準條例第1條前段、
第2條，現行法規所定貨幣單位折算新臺幣條例第2條，判決如主
文。

│    │        │  修正時，刑法分則編未修正之條文定有罰金者，自94年1 │
│    │        │  月7 日刑法修正施行後，就其所定數額提高為30倍。但  │
│    │        │  72年6 月26日至94年1 月7 日新增或修正之條文，就其所│
│    │        │  定數額提高為3 倍。是以刑法之貨幣單位由「元（指銀元│
│    │        │  ）」變更為「新臺幣」，且刑法分則之罰金數額，亦視該│
│    │        │  分則先前曾修正與否，而分別提高3 或30倍。經按現行法│
│    │        │  規所定貨幣單位折算新臺幣條例第2 條之規定折算後等值│
│    │        │  ，故新法並未較有利。                              │
│    │        │(二)刑法第33條第5 款有關罰金刑部分，在新法部分係規定  │
│    │        │  為罰金：新臺幣1,000 元以上，以百元計算之。而舊法  │
│    │        │  時則係規定為罰金：1 元以上，修正後之規定並非較有  │
│    │        │  利於被告。                                        │
├──┼────┼──────────────────────────┤
│ 2  │ 幫助犯 │刑法第30條有關幫助犯之規定，僅作文字修正，無關成立幫│
│    │        │助犯內容之實質變更，非屬法律之變更。                │
├──┴────┴──────────────────────────┤
│經綜合比較新、舊法之結果，以被告丁○○行為時之舊法較有利於被告，依刑│
│法第2條第1項前段規定，本案關於被告丁○○部分自應一體適用被告行為時之│
│法律即修正前第33條第5款、罰金罰鍰提高標準條例第1條前段等規定。      │
└──────────────────────────────────┘
附表3
┌─────┬─────────────────────────┐
│修正事項  │新、舊法之比較適用                                │
├─────┼─────────────────────────┤
│易科罰金  │刑法第41條第1項前段關於得易科罰金之規定，修正前係 │
│          │以「因身體、教育、職業、家庭之關係或其他正當事由，│
│          │執行顯有困難」為其要件，新法則刪除此要件，修正理由│
│          │說明上述規定限制過嚴，欠缺彈性，與易科罰金制度旨在│
│          │救濟短期自由刑流弊之旨趣不符，且刪除該要件後使檢察│
│          │官得視具體個案審酌宜否易科罰金，更符告易科罰金制度│
│          │之意旨。另修正前刑法第41條第1項前段配合罰金罰鍰提 │
│          │高標準條例第2條（現已刪除）之規定，係以「銀元300元│
│          │即新臺幣900元折算1日」為易科罰金之折算標準，至修正│
│          │後之刑法第41條第1項乃係以「新臺幣1,000元、2,000元 │
│          │或3,000元折算1日」為易科罰金之折算標準。而觀諸歷年│
│          │來之實務運作情形，在法院宣告易科罰金折算標準之案件│
│          │，檢察官執行時乃以准予易科罰金為原則，不准易科罰金│
│          │為例外，是以比較修法前後就易科罰金准否及折算標準之│
│          │規定，應認修正後刑法第41條第1項前段提高易科罰金之 │
│          │折算標準之適用，對於行為人而言較為不利。是依刑法第│
│          │2 條第1 項前段，應適用修正前刑法第41條第1 項前段之│
│          │規定。                                            │
