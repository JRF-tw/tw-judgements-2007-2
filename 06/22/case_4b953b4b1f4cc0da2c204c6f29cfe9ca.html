臺灣彰化地方法院刑事判決　　　　　　　 96年度訴字第347號
公　訴　人　臺灣彰化地方法院檢察署檢察官
被　　　告　辛○○
指定辯護人　本院公設辯護人　己○○
上列被告因違反毒品危害防制條例案件，經檢察官提起公訴（96
年度偵字第834 號、第1673號），本院判決如下：
    主  文
辛○○連續販賣第一級毒品，累犯，處有期徒刑拾貳年陸月，因
販賣第一級毒品所得新臺幣肆萬貳仟伍佰元沒收之，如全部或一
部不能沒收時，以其財產抵償之，未扣案原內裝門號00000
00000門號之行動電話壹支沒收，如全部或一部不能沒收時
，追徵其價額或以其財產抵償之；又連續販賣第二級毒品，累犯
，處有期徒刑柒年貳月，因販賣第二級毒品所得新臺幣肆仟元沒
收之，如全部或一部不能沒收時，以其財產抵償之，未扣案原內
裝門號0000000000門號之行動電話壹支沒收，如全部
或一部不能沒收時，追徵其價額或以其財產抵償之；又連續轉讓
第一級毒品，累犯，處有期徒刑壹年肆月，未扣案原內裝門號0
000000000門號之行動電話壹支沒收，如全部或一部不
能沒收時，追徵其價額或以其財產抵償之；又連續明知為禁藥而
轉讓，累犯，處有期徒刑捌月，未扣案原內裝門號000000
0000門號之行動電話壹支沒收。應執行有期徒刑拾玖年陸月
，因販賣第一級毒品所得新臺幣肆萬貳仟伍佰元及販賣第二級毒
品所得新臺幣肆仟元沒收之，如全部或一部不能沒收時，以其財
產抵償之，未扣案原內裝門號0000000000門號之行動
電話壹支沒收，如全部或一部不能沒收時，追徵其價額或以其財
產抵償之。
    犯罪事實
一、辛○○（綽號紅毛、阿毛）前曾於民國85年間，因恐嚇案件
    ，經本院以85年度易字第2625號，判處有期徒刑1 年6 月，
    緩刑5 年確定；後於緩刑期內之87年間，因竊盜案件，經本
    院以88年度易字第273 號，判處有期徒刑1 年4 月確定；復
    於88年間，因施用第一、二級毒品案件，經本院以88年度訴
    字第381 號，分別判處有期徒刑7 月、5 月確定，前開3 罪
    定應執行有期徒刑2 年確定，且上開恐嚇案件之緩刑亦經撤
    銷，入監接續執行後，於91年4 月22日因縮短刑期假釋出監
    ，且於92年6 月9 日假釋期滿未經撤銷，以已執行完畢論。
    詎其仍不知悔改，因有施用毒品之慣行，明知海洛因業經毒
    品危害防制條例列為第一級毒品管制，依法不得販賣、轉讓
    、持有，且甲基安非他命除業經毒品危害防制條例列為第二
    級毒品管制，而「安非他命類」 (Amphetamine-li ke） 藥
    品，前經行政院衛生署於69年12月8 日衛署藥字第301124號
    公告禁止輸入、製造、販賣，而屬於藥事法第22條第1 款所
    規定之禁藥，嗣經同署於79年10月9 日以衛署藥字第904142
    號公告列入麻醉藥品管理條例第2 條第4 款所定「化學合成
    麻醉藥品」管理，惟仍不失其為禁藥之性質，不得非法販賣
    、轉讓及持有，詎其仍為下列之行為：
  (一)為圖謀厚利俾供購毒施用，竟意圖營利，基於販賣第一級毒
    品海洛因之概括犯意，先在臺灣地區某不詳地點，向不詳姓
    名年籍之成年人購入不詳重量之第一級毒品海洛因後，即於
    附表壹所示之交易時間、交易地點，以附表壹所示之交易方
    式，連續以附表壹所示之交易價格，販賣第一級毒品海洛因
    予丙○○、庚○○、戊○○、乙○○等人，販賣海洛因所得
    之款項共計新臺幣（下同）42,500，則留供己充作購毒之用
    。
  (二)為圖謀厚利俾供購毒施用，竟復另行意圖營利，基於販賣第
    二級毒品甲基安非他命之概括犯意，先在臺灣地區某不詳地
    點，向不詳姓名年籍之成年人購入不詳重量之第二級毒品甲
    基安非他命後，即於附表貳所示之交易時間、交易地點，以
    附表貳所示之交易方式，連續以附表貳所示之交易價格，販
    賣第二級毒品甲基安非他命予丁○○及甲○○各2 次，販賣
    甲基安非他命所得之款項共計4,000 元。
  (三)辛○○於與庚○○因販賣海洛因熟識後，基於朋友情誼，竟
    基於轉讓第一級毒品海洛因之概括犯意，先後於附表所示
    之轉讓時間、轉讓地點，以附表所示之約定方式聯絡見面
    後，即以無償之方式，連續轉讓第一級毒品海洛因予庚○○
    11次（每次轉讓之量微，均僅供單次施用，且11次之數量合
    計淨重未超過5 公克）。
  (四)辛○○於與甲○○因販賣甲基安非他命熟識後，基於朋友情
    誼，竟基於轉讓屬禁藥之甲基安非他命之概括犯意，先後於
    附表肆所示之轉讓時間、轉讓地點，以附表肆所示之約定方
    式聯絡見面後，即以無償之方式，連續轉讓屬禁藥之甲基安
    非他命予甲○○2 次（每次轉讓之量微，均僅供單次施用，
    且2 次之數量合計淨重未超過10公克）。
    嗣經臺灣彰化地方法院檢察署另案指揮司法警察偵查巫有生
    違反毒品危害防制條例案件，而實施通訊監察時，察覺辛○
    ○前開販賣第一級毒品海洛因、第二級毒品甲基安非他命之
    犯罪嫌疑，循線追查後，另查獲辛○○前開轉讓第一級毒品
    海洛因及轉讓屬禁藥之甲基安非他命犯行。
二、案經彰化縣警察局芳苑分局報告臺灣彰化地方法院檢察署檢
    察官偵查起訴。
    理  由
壹、有關於證據能力部分：
    本案全部卷證所涵括之供述證據及非供述證據，經本院為被
    告辛○○所指定之本院公設辯護人閱卷後，公訴人、被告及
    辯護人均同意全部證據之證據能力，是後述所引用證據之證
    據能力均無疑義，合先敘明。
貳、有關於認定犯罪事實：
    訊據被告辛○○就其於前開時間、地點，販賣第一級毒品海
    洛因予證人丙○○、庚○○、戊○○、乙○○等人；及於前
    開時間、地點，販賣第二級毒品甲基安非他命予證人甲○○
    及丁○○；及於前開時間、地點，無償轉讓第一級毒品海洛
    因予證人庚○○；及於前開時間、地點，無轉轉讓屬禁藥之
    甲基安非他命予證人甲○○等情均坦承不諱，核與證人丙○
    ○、戊○○、乙○○於本院審理時證述向被告購買海洛因之
    情節，及證人庚○○於本院審理時，證述向被告購買海洛因
    及無償受讓海洛因之情節，及證人甲○○於本院審理時，證
    述向被告購買甲基安非他命及無償受讓甲基安非他命之情節
    ，及證人丁○○於警詢及檢察官偵訊時，證述向被告購買甲
    基安非他命之情節均相符（詳見本院審判筆錄及96年度偵字
    第834 號卷第14頁、第25頁），此外，並有被告持用之門號
    000000 0000 行動電話之通訊監察譯文摘要附卷可稽（詳見
    96年度偵字第834 號卷第16頁至第17頁、第96頁至第112 頁
    、第132 頁、第138 頁至第139 頁、第158 頁至第171 頁）
    ，足見被告上開自白確與事實相符。又參以近年來毒品之濫
    用，危害國民健康與社會安定日益嚴重，治安機關對於販賣
    或施用毒品之犯罪行為，無不嚴加查緝，各傳播媒體對於政
    府大力掃毒之決心亦再三報導，已使毒品不易取得且物稀價
    昂，苟被告於買賣之過程中無利可圖，縱屬至愚，亦無甘冒
    被取締移送法辦判處重刑之危險而平白從事海洛因及甲基安
    非他命買賣之理，且被告於本院審理時亦自承於向上手販入
    海洛因及甲基安非他命後，會先自行留取部分海洛因及甲基
    安非他命後，再將所剩餘之海洛因及甲基安非他命販出等語
    （詳見本院審判筆錄），是被告於附表壹、貳所示之時間、
    地點，販賣海洛因及甲基安非他命予附表壹、貳所示之丙○
    ○、庚○○、戊○○、乙○○、丁○○及甲○○等人際，有
    以留扣部分海洛因、甲基安非他命之量差方式以牟利之意圖
    及事實，應可認定。至證人丁○○於本院審理時，雖翻異前
    詞，改證稱其未曾在附表貳編號1 所示之95年2 月間，與被
    告碰面購買甲基安非他命等語（見本院審判筆錄），惟證人
    丁○○於警詢時，即已就其向被告購買甲基安非他命之時間
    、地點證述綦詳，復於員警依照通訊監察譯文內容，詢問該
    疑似毒品交易之通訊監察譯文是否即為與被告間之毒品交易
    聯絡時，證人丁○○復清楚地向司法警察證稱因被告販售價
    格較貴，故該2 次通訊監察譯文所示之交易並未成功；而於
    同日接受檢察官偵訊時，證人丁○○亦證稱確曾跟被告購買
    甲基安非他命2 次等情（詳見96年度偵字第834 號卷第14頁
    、第25頁），而證人丁○○於本院審理時，經本院提示其前
    開警、偵訊筆錄供其閱讀並逐字朗讀筆錄後，證人丁○○亦
    證稱筆錄記載之內容，確實均係其之前所為之陳述，應可認
    證人丁○○於警、偵訊時之證述，已具有相當高之憑信性；
    況證人丁○○於本院審理時證述未曾向被告購買甲基安非他
    命後，經本院詢問被告意見，被告仍明確陳述確曾販賣甲基
    安非他命2 次予證人丁○○等語（詳見本院審判筆錄），綜
    前所述，應足認證人丁○○於本院審理時之證述，應與事實
    不符，尚無從採信。綜上，本案事證均已明確，被告販賣第
    一級毒品、第二級毒品，轉讓第一級毒品，轉讓禁藥甲基安
    非他命之犯行，均堪認定，應予依法論科。
、有關論罪科刑部分：
一、我國刑法於94年2 月2 日修正公布，95年7 月1 日施行（下
    稱新法；修正前刑法下稱舊法）後，依最高法院95年度第 8
    次刑事庭會議決議內容，有關新舊法之適用原則如下：
    (一)新法第2條第1項之規定，係規範行為後法律變更所生新舊
      法律比較適用之準據法，於新法施行後，應適用新法第 2
      條第1項之規定，為「從舊從輕」之比較。
    (二)基於罪刑法定原則及法律不溯及既往原則，行為之處罰，
      以行為時之法律有明文規定者為限，必行為時與行為後之
      法律均有處罰之規定，始有新法第2條第1項之適用。
    (三)比較時應就罪刑有關之共犯、未遂犯、想像競合犯、牽連
      犯、連續犯、結合犯，以及累犯加重、自首減輕暨其他法
      定加減原因（如身分加減）與加減例等一切情形，綜其全
      部罪刑之結果而為比較。
    (四)新法第33條第5款規定罰金刑為新臺幣1,000元以上，以百
      元計算之，新法施行後，應依新法第2條第1項之規定，適
      用最有利於行為人之法律。
    (五)新法第51條第5 款提高多數有期徒刑合併應執行之刑不得
      逾30年，新法施行後，應依新法第2 條第1 項之規定，適
      用最有利於行為人之法律。
    (六)新法施行前，犯法定本刑為死刑、無期徒刑之罪，有減輕
      其刑之原因者，新法施行後，應依新法第2 條第1 項之規
      定，適用最有利於行為人之法律。
    (七)新法施行前，法定罰金刑有加減之原因者，新法施行後，
      應依新法第2 條第1 項之規定，適用最有利於行為人之法
      律。
    本案被告行為後，刑法關於連續犯、累犯、主刑為罰金刑、
    刑之減輕、酌減、有期徒刑定應執行等部分之條文，已有所
    修正，並自95年7 月1 日開始施行。本院依前揭最高法院刑
    事庭會議之決議，就被告涉及犯行之相關論罪科刑條文，為
    如附表伍所示之綜合比較後，認應適用修正前刑法有關連續
    犯、累犯及刑之減輕、酌減、定應執行等規定。
二、按海洛因及甲基安非他命分係毒品危害防制條例所定之第一
    級及第二級毒品，被告基於營利之目的，以賺取量差之方式
    ，販賣第一級毒品海洛因予附表壹所示之丙○○、庚○○、
    戊○○、乙○○，及販賣第二級毒品甲基安非他命予附表貳
    所示之丁○○及甲○○，核被告此部分所為，分別係犯毒品
    危害防制條例第4 條第1 項之販賣第一級毒品既遂罪及同條
    第2 項之販賣第二級毒品既遂罪。起訴書之犯罪事實欄一(五)
    、(六)業已載明被告有為附表貳所示之販賣第二級毒品罪行，
    惟於證據並所犯法條欄漏載被告此部分所為，有違反毒品危
    害防制條例第4 條第2 項之販賣第二級毒品既遂罪，僅屬法
    條漏載，本院自應審理，併予敘明。又被告無償轉讓第一級
    毒品海洛因予附表所示之庚○○（總轉讓克數合計淨重未
    超過5 公克），核其此部分所為，係犯毒品危害防制條例第
    8 條第1 項之轉讓第一級毒品罪。再按「安非他命類」（Am
    phet amine-like） 藥品，前經行政院衛生署於69年12月 8
    日衛署藥字第301124號公告禁止輸入、製造、販賣，而屬於
    藥事法第22條第1 款所規定之禁藥，嗣經同署於79年10月 9
    日以衛署藥字第904142號公告列入麻醉藥品管理條例第2 條
    第4 款所定「化學合成麻醉藥品」管理，惟仍不失其為禁藥
    之性質，不得非法持有及轉讓。又藥事法業已於93年4 月21
    日修正公布生效，修正後藥事法第83條第1 項規定，明知為
    偽藥或禁藥，而販賣、供應、調劑、運送、寄藏、牙保、轉
    讓或意圖販賣而陳列者，處7 年以下有期徒刑，得併科新臺
    幣5 百萬元以下罰金；又查肅清煙毒條例名稱業於87年5 月
    20日修正為毒品危害防制條例，並於同日修正公布全文，同
    年月22日起生效施行。於該條例中將毒品依其成癮性、濫用
    性及對社會危害性分為三級，而甲基安非他命則明定列為第
    二級毒品，轉讓第二級毒品者，同條例第8 條第2 項訂有處
    罰之明文。毒品危害防制條例嗣又修正，於92年7 月9 日公
    布，93年1 月9 日施行，其中第8 條增訂第6 項「轉讓毒品
    達一定數量者，加重其刑至2 分之1 ，其標準由行政院定之
    。」之加重處罰規定。蓋毒品之範圍尚包括影響精神物質與
    其製品。而藥事之管理，亦非僅止於藥品之管理。又毒品未
    必係經公告之禁藥，禁藥亦未必為毒品。毒品危害防制條例
    與藥事法二者，並無必然之特別法與普通法關係。從而，如
    所轉讓之第二級毒品同時亦為藥事法所公告之禁藥。且為行
    為人所明知，則除有轉讓之第二級毒品達毒品危害防制條例
    第8 條第6 項之一定數量，經依法加重後之法定刑較藥事法
    第83條第1 項之法定刑為重之情形外，因藥事法第83條第 1
    項為後法，且為重法，應優先適用藥事法處罰。從而，被告
    明知甲基安非他命為禁藥，而無償轉讓合計淨重未超過10
    公克之甲基安非他命予附表肆所示之甲○○，核其此部分所
    為，係犯藥事法第83條第1 項之明知為禁藥而轉讓既遂罪。
    又被告販賣第一級毒品海洛因、轉讓第一級毒品海洛因前持
    有第一級毒品之低度行為，應各為販賣第一級毒品及轉讓第
    一級毒品之高度行為所吸收，均不另論罪。再被告販賣第二
    級毒品甲基安非他命前持有第二級毒品之低度行為，亦應為
    販賣第二級毒品之高度行為所吸收，亦不另論罪。另被告轉
    讓禁藥甲基安非他命（甲基安非他命亦屬第二級毒品）前持
    禁藥甲基安非他命之低度行為，應為轉讓禁藥之高度行為所
    吸收，亦不另論以毒品危害防制條例第11條第2 項之持有第
    二級毒品罪。再被告先後23次販賣第一級毒品（丙○○11
    次、庚○○10次、戊○○1 次、乙○○1 次）、4 次販賣第
    二級毒品（丁○○2 次、甲○○2 次）、11次轉讓第一級毒
    品，及2 次轉讓禁藥之犯行，均時間緊接，各犯同一罪名之
    罪，顯係各基於概括犯意反覆為，均應依修正刪除前連續犯
    之規定，各論以販賣第一級毒品既遂一罪、販賣第二級毒品
    既遂一罪、轉讓第一級毒品既遂一罪，及明知為禁藥而轉讓
    既遂一罪，並均依法加重其刑（其中因販賣第一級毒品罪之
    法定本刑為死刑、無期徒刑，被告此部分依法不得再予加重
    ，僅就罰金刑部分加重；又就販賣第二級毒品罪之法定本刑
    無期徒刑部分，依法亦不得加重，僅就其餘部分加重之）。
    又公訴人就附表壹編號1 有關被告販賣第一級毒品予丙○○
    之另1 次犯行（起訴書僅起訴販賣予丙○○10次），及附表
    編號1 有關被告轉讓第一級毒品予庚○○另1 次之犯行（
    起訴書僅起訴轉讓10次），雖均未據公訴人提起公訴，惟此
    部分販賣第一級毒品海洛因及轉讓第一級毒品海洛因之犯行
    ，與檢察官起訴被告連續販賣第一級毒品及連續轉讓第一級
    毒品海洛因部分，既分有修正刪除前連續犯之裁判上一罪關
    係，應均為起訴效力所及，本院自得併予審理，附此敘明。
    再被告所犯上開毒品危害防制條例第4 條第1 項之販賣第一
    級毒品既遂罪、第4 條第2 項之販賣第二級毒品既遂罪、第
    8 條第1 項之轉讓第一級毒品既遂罪、藥事法第83條第1項
    之明知為禁藥而轉讓既遂罪4 罪間，犯意各別，罪質互殊，
    應予分論併罰。另被告前曾於85年間，因恐嚇案件，經本院
    以85年度易字第2625號，判處有期徒刑1 年6 月，緩刑5 年
    確定；後於緩刑期內之87年間，因竊盜案件，經本院以88年
    度易字第273 號，判處有期徒刑1 年4 月確定；復於88年間
    ，因施用第一、二級毒品案件，經本院以88年度訴字第 381
    號，分別判處有期徒刑7 月、5 月確定，前開3 罪定應執行
    有期徒刑2 年確定，且上開恐嚇案件之緩刑亦經撤銷，入監
    接續執行後，於91年4 月22日因縮短刑期假釋出監，且於92
    年6 月9 日假釋期滿未經撤銷，以已執行完畢論等情，有臺
    灣高等法院被告前案紀錄表存卷可參，其於5 年以內再犯本
    件有期徒刑以上之4 罪，均為累犯，皆應依修正前刑法第47
    條規定遞加重其刑（因販賣第一級毒品罪之法定本刑為死刑
    、無期徒刑，被告此部分依法不得再予加重，僅就罰金刑部
    分遞加重；又就販賣第二級毒品罪之法定本刑無期徒刑部分
    ，依法亦不得遞加重，僅就其餘部分遞加重之）。按販賣第
    一級毒品罪之法定刑為死刑或無期徒刑，然同為販賣第一級
    毒品之人，其原因動機不一，犯罪情節未必盡同，或有大盤
    毒梟者，亦有中、小盤之分，甚或僅止於吸毒者友儕間為求
    互通有無之有償轉讓者亦有之，其販賣行為所造成危害社會
    之程度自屬有異，法律科處此類犯罪，所設之法定最低本刑
    卻同為「死刑或無期徒刑」，不可謂不重。於此情形，倘依
    其情狀處以較輕之徒刑，即足以懲儆，並可達防衛社會之目
    的者，自非不可依客觀之犯行與主觀之惡性二者加以考量其
    情狀，是否有可憫恕之處，適用刑法第59條之規定酌量減輕
    其刑，期使個案裁判之量刑，能斟酌至當，符合比例原則。
    本件被告先後多次販賣第一級毒品海洛因，無視國家對於杜
    絕毒品危害之禁令，其行為固屬不當，應予非難，然其販賣
    毒品對象尚僅有證人丙○○、庚○○、戊○○及乙○○，而
    該等之人均係因與被告同染有毒癮之友人，並非與其不認識
    之初用毒品者，被告前後販賣海洛因之數量，依其價錢估算
    ，亦非至鉅，其犯罪之情節尚非至惡，被告僅因一時貪念，
    致罹重典，然相對於長期大量販賣毒品之大毒梟而言，其對
    社會治安及國民健康之危害顯然較小，且被告前雖有上揭前
    科犯行，然尚查無販賣、意圖販賣而持有毒品等前科素行，
    綜上被告犯案情節觀之，倘仍遽處以法定本刑之最低刑度即
    無期徒刑，無異仍屬失之過苛而不盡情理，不免予人情輕法
    重之感，且難謂符合罪刑相當性及比例原則，更無從與大毒
    梟之惡行有所區隔，是其犯罪情狀相較於法定之重刑，在客
    觀上足以引起一般人之同情，情節尚堪憫恕，爰依修正前刑
    法第59 條 之規定，對被告所犯販賣第一級毒品海洛因之犯
    行，酌量減輕其刑，並依修正前規定就法定刑死刑減為無期
    徒刑或15 年 以下12年以上有期徒刑，無期徒刑則減為7 年
    以上有期徒刑，至法定罰金刑部分，則就最高度先加後減輕
    其刑。爰審酌被告平日素行不良、犯罪之動機、目的係為圖
    得一己之厚利、販賣第一級毒品、第二級毒品期間之長短、
    次數、轉讓第一級毒品之次數、期間長短、轉讓禁藥甲基安
    非他命之次數、期間長短，販賣、轉讓毒品及禁藥均係增進
    流通毒品、禁藥之管道對社會治安、經濟所生危害甚巨，惟
    被告犯罪後對全部犯行均坦承不諱等一切情狀，分別量處如
    主文所示之刑，並依修正前第51條第5 款規定，定其應執行
    之刑。
四、另按毒品危害防制條例第19條第 1項規定，犯販賣毒品罪者
    ，其犯罪所得之財物均沒收之，係採義務沒收主義。故販賣
    毒品所得之對價，不問其中成本若干，利潤多少，均應全部
    諭知沒收，貫徹政府查禁煙毒之決心，以符立法本旨（最高
    法院91年度臺上字第2419號判決意旨參照）。查公訴人雖認
    定被告販賣海洛因予證人丙○○之金額係5,000 元至18,000
    元不等，然證人丙○○於本院審理時，已具結證稱其購買海
    洛因之價格應係5,000 元2 次、3,000 元3 次及2,000 元 2
    次等語，衡之證人丙○○於本院審理時就各次購買金額，已
    做具體描述，相較於其於警詢、偵訊時，僅就購買海洛因之
    金額籠統回答，應以於本院審理時謹慎思考後，所為有利於
    被告之前開陳述較為可採。是被告販賣第一級毒品海洛因予
    丙○○之所得為31,000元，販賣第一級毒品海洛因予庚○○
    之所得為10,000元，販賣第一級毒品海洛因予戊○○之所得
    為1,000 元，販賣第一級毒品海洛因予乙○○之所得為 500
    元；另販賣第二級毒品甲基安非他命予丁○○之所得為2,00
    0 元，販賣第二級毒品甲基安非他命予甲○○之所得為2,00
    0 元，上開被告販賣第一級毒品海洛因及販賣第二級毒品甲
    基安非他命所得，雖均未扣案，仍應依上開說明及毒品危害
    防制條例第19條第1 項之規定宣告沒收之，如全部或一部不
    能沒收時，則均以被告之財產抵償之。另供作被告販賣第一
    級毒品海洛因、第二級毒品甲基安非他命及轉讓第一級毒品
    海洛因之原內含0000000000門號SIM 卡之行動電話1 支，係
    被告所有，業據被告自承在卷，雖未扣案，仍應依同條例第
    19 條 第1 項規定併予宣告沒收，如全部或一部不能沒收時
    ，應追徵其價額或以其財產抵償之。再前開行動電話1 支，
    亦係被告所有，供其聯絡轉讓屬禁藥之甲基安非他命所用之
    工具，業據被告自承在卷，亦應依刑法第38條第1 項第2 款
    規定，宣告沒收。至該具行動電話內之SIM 卡，係被告之不
    知情友人武氏藏借予被告使用，尚非被告本人親自申請等情
    ，業據被告及證人武氏藏陳稱在卷，故該SIM 卡非屬被告所
    有，而毒品危害防制條例第19條第1 項規定宣告沒收，因該
    條項並無如同條例第18條第1 項所定（不問屬於犯人所有與
    否，沒收之）之明文，故應屬相對沒收主義之立法，其應沒
    收之物，應以屬被告所有者為限（最高法院93年度臺上字第
    46 2 號 判決參照），而刑法第38條第1 項第2 款規定，供
    被告犯罪所用之物，亦以該物品係被告所有為前提，準此，
    上開SIM 卡被告既未取得所有權，自毋庸依毒品危害防制條
    例第19條第1 項規定及刑法第38條第1 項第2 款規定宣告沒
    收之，併此敘明。
五、公訴意旨雖以被告基於販賣第一級毒品海洛因之犯意，於95
    年4 月間某日，在被告當時位於彰化縣員林鎮○○路35 號6
    樓之7 住處，以2,000 元之價格，販賣第一級毒品海洛因予
    證人戊○○1 次，而認被告此部分亦涉有販賣第一級毒品海
    洛因罪嫌。然按犯罪事實應依證據認定之，無證據不得認定
    犯罪事實，刑事訴訟法第154 條第2 項定有明文。而認定不
    利於被告之事實，須依積極證據，苟積極證據不足為不利於
    被告事實之認定時，即應為有利於被告之認定，更不必有何
    有利之證據，亦有最高法院30年上字第816 號判例可資參照
    。經查：證人戊○○於本院審理時，具結後證稱：其僅跟被
    告買過1 次海洛因，後來都是合資的等語（詳見本院審判筆
    錄），證人戊○○此部分向被告購買海洛因次數之證述，雖
    與其於偵訊時證述不一，然證人戊○○於本院交互詰問下，
    深思熟慮後，就交易次數所為較為保守、謹慎之證述，其可
    信度頗高；且就被告另有於95年4 月間某日，在被告當時住
    處，以2,000 元之價格，販賣第二級毒品甲基安非他命予證
    人戊○○1 次之犯行，因證人戊○○業已翻異前詞，卷內亦
    無其他積極證據足資證明，基於罪疑唯輕法理，依照現有積
    極證據，尚無從證明被告另有於95年4 月間某日，在被告當
    時住處，另行販賣第一級毒品海洛因予證人戊○○之犯行，
    被告此部分被訴犯罪自屬不能證明，揆諸首揭法條及判例意
    旨說明，此部分本應為被告無罪之諭知，惟因公訴意旨認此
    部分與被告前揭經論罪科刑之販賣第一級毒品犯行部分，有
    修正刪除前連續犯之裁判上一罪關係，爰不另為無罪之諭知
    ，附此說明。

法第83條第1 項，刑法第11條、第2 條第1 項前段、修正刪除前
刑法第56條、修正前刑法第47條、第59條、第51條第5 款、第38

附表伍
┌──┬─────┬─────────────────────────┐
│編號│修正事項  │         新、舊法之比較適用                       │
├──┼─────┼─────────────────────────┤
│ 1  │連續犯    │被告行為後，刑法第56條連續犯之規定，業經刪除，則被│
│    │          │告之犯行，因行為後新法業已刪除連續犯之規定，此刪除│
│    │          │雖非犯罪構成要件之變更，但已顯已影響行為人刑罰之法│
│    │          │律效果，自屬法律有變更，且比較新、舊法結果，以舊法│
│    │          │論以連續犯對被告較為有利                          │
├──┼─────┼─────────────────────────┤
│ 2  │累犯      │刑法第47條原規定：「受有期徒刑之執行完畢，或受無期│
│    │          │徒刑或有期徒刑之執行而赦免後，5 年以內再犯有期徒刑│
│    │          │以上之罪者，為累犯，加重本刑至2 分之1 。」，修正後│
│    │          │第47條第1 項規定：「受徒刑之執行完畢，或一部之執行│
│    │          │而赦免後，5 年以內故意再犯有期徒刑以上之罪者，為累│
│    │          │犯，加重本刑至2 分之1 」。被告本案所犯係故意為之，│
│    │          │其修正前後結果並無不同                            │
├──┼─────┼─────────────────────────┤
│ 3  │刑之酌減  │刑法第59條原規定：「犯罪之情狀可憫恕者，得酌量減輕│
│    │          │其刑」，修正後第59條規定：「犯罪之情狀顯可憫恕，認│
│    │          │科以最低刑仍嫌過重者，得酌量減輕其刑」。觀諸該條修│
│    │          │正理由，修正目的係為強調「可憫恕情狀較為明顯」之要│
│    │          │件，以防止刑之酌減遭濫用，且將實務上向來所持「認縱│
│    │          │予宣告法定最低刑度猶嫌過重」之見解，使其明文化。綜│
│    │          │合比較修正前、後之條文，其修正前後結果並無不同    │
├──┼─────┼─────────────────────────┤
│ 4  │死刑之減輕│修正前刑法第64條規定：「死刑減輕者，為無期徒刑，或│
│    │          │為15年以下12年以上有期徒刑」，修正後第64條規定：「│
│    │          │死刑減輕者，為無期徒刑」，經比較修正前、後之規定，│
│    │          │應以修正前之規定較有利於行為人                    │
├──┼─────┼─────────────────────────┤
│ 5  │無期徒刑之│修正前第65條規定：「無期徒刑減輕者，為7 年以上有期│
│    │減輕      │徒刑」，修正後第65條規定：「無期徒刑減輕者，為20年│
│    │          │以下15年以上有期徒刑」，經比較修正前、後之規定，應│
│    │          │以修正前之規定較有利於行為人                      │
├──┼─────┼─────────────────────────┤
│ 6  │主刑罰金刑│刑法第33條第5款有關罰金刑部分，在新法部分係規定為 │
│    │部分      │罰金：新臺幣1,000元以上，以百元計算之。而舊法時則 │
│    │          │係規定為罰金：1元以上。經比較修法前後之規定，應認 │
│    │          │修正前刑法第33條第5款之規定，對行為人有利。       │
├──┼─────┼─────────────────────────┤
│ 7  │罰金之減輕│刑法修正前對罰金之減輕並未規定，乃因修正前刑法第33│
│    │          │條規定罰金為1 元以上，倘減輕之，即可能造成不滿1 元│
│    │          │之零數，惟修正後刑法第33條已明定罰金為新台幣1 千元│
│    │          │以上，當不致因減輕其最低度刑，而產生不滿1 元之零數│
│    │          │，是以修正後第69條規定罰金之加減，其最高及最低度刑│
│    │          │同加減之。比較修正前、後之規定，應以修正前之規定較│
│    │          │有利於行為人                                      │
├──┼─────┼─────────────────────────┤
│ 8  │定應執行刑│關於刑法第51條，依舊法第5 款之規定，宣告多數有期徒│
