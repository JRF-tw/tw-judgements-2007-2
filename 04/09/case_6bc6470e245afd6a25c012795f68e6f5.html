【裁判字號】 95,簡,695
【裁判日期】 960409
【裁判案由】 綜合所得稅
【裁判全文】   
臺北高等行政法院判決
　　　　　　　　　　　　　　　　　　 95年度簡字第00695號
原　　　告　甲○○
　　　　　　送達代收人　乙○○.
被　　　告　財政部臺北市國稅局
代　表　人　許虞哲（局長）住同上
上列當事人間因綜合所得稅事件，原告不服財政部中華民國95年
8 月17日台財訴字第09500325490 號（案號：第00000000號）訴
願決定，提起行政訴訟。本院判決如下：
　　主  文
原告之訴駁回。
訴訟費用由原告負擔。
　　事  實
一、事實概要：原告民國（下同）93年度綜合所得稅結算申報，
    經財政部臺灣省中區國稅局（下稱中區國稅局）台中市分局
    查得其將坐落台中市○區市○路58號房屋（應有部分1/2 ，
    下稱系爭房屋）出租予星喬美容美髮材料行（下稱星喬行）
    ，依查得資料及當地一般租金標準，核定當年度租賃所得為
    新台幣（下同）47,951元，並通報被告併課核定當年度綜合
    所得總額為2,381,521 元，淨額為1,705,452 元。原告不服
    ，申請復查，經被告95年5 月29日財北國稅法字第09502030
    76號復查決定駁回，原告仍不服，向財政部提起訴願，亦遭
    決定駁回，遂向本院提起本件行政訴訟。
二、兩造聲明：
  (一)原告聲明：撤銷原處分。
　(二)被告聲明：駁回原告之訴。
三、兩造之爭點：
    被告設算核定系爭房屋之租金，於法是否有據？
　(一)原告主張之理由：
    1.由於星喬行從未有租金費用之支出，可充分證明原告未曾
      因系爭房屋發生任何所得。原告91年綜合所得稅亦有相同
      案情，已經鈞院（94年簡字第441 號）查明案情，並撤銷
      該原處分在案。
    2.無所得者即無繳納所得稅之義務，政府無權假借任何名義
      對其課徵所得稅，此為放諸四海皆準神聖不可侵犯的基本
      原理。且所得稅非屬行為稅，不能只因有某種行為而以設
      算方式課徵之，其特性是要個人或法人因某些行為之「結
      果」，實際發生所得才須繳納之稅負。實際發生的所得是
      課徵所得稅決定必要的先決條件，未曾發生所得者不受所
      得稅法之規範；換言之，所得稅法的法條不論訂定得多巧
      妙或荒謬，其效力絕不能即於無所得者，此為基本之法理
      。前財政部長林全亦於93年6 月透過所有媒體公開表達此
      一原理。
    3.被告主張人民應繳納所得稅，依法應負舉證責任，證明人
      民發生所得之事實，本有最高行政法院之判例可為遵循。
      又現時電腦資訊作業極為發達，被告稅捐資料建檔完備，
      相關資料的查證易如反掌，被告實無理由或藉口可免除該
      舉證責任。
    4.系爭房屋係由原告與弟林育樹各以二分之一共同持有，依
      民法第818 條規定，各共有人均對該房屋有全部使用權，
      此為法律特別規定。林育樹使用系爭房屋經營星喬行，依
      法無須事前與其他共有人取得任何形式的協議，因此完全
      不發生借用的問題。
    5.星喬行經營績效偏低，即使不計房屋及人事費用，其所得
      產生之收益遠低於政府所定最低工資標準。縱使以所謂內
      部關係論，星喬行雖使用原告持分房屋，實際上亦無法分
      取任何收益。
    6.由於星喬行未曾有房租及人事費用之支出，是故系爭房屋
      所能產生之收益已全部反映於該行之報稅資料中，應納的
      稅均未少繳，因此原告絕無再行繳納任何所得稅之理。
    7.系爭房屋位處台中市中區商圈，商機日趨沒落，鄰近店家
      連續5 年以上未使用比比皆是，該地區的政府公告地價與
      房屋稅均陸續下調，鄰近店家實際已不具出租價值。
    8.原告自87年起，每年申請復查綜合所得稅，其情形如下：
      ┌──┬──────┬─────┬─────┐
      │年度│ 中區國稅局 │復查決定數│ 核減比率 │
      │    │ 通  報  數 │          │          │
      ├──┼──────┼─────┼─────┤
      │ 87 │  83,577元  │ 13,788元 │  83.50%  │
      ├──┼──────┼─────┼─────┤
      │ 88 │  83,577元  │ 13,788元 │  83.50%  │
      ├──┼──────┼─────┼─────┤
      │ 89 │  82,162元  │ 13,492元 │  83.58%  │
      ├──┼──────┼─────┼─────┤
      │ 90 │  68,502元  │ 12,032元 │  82.44%  │

      │ 91 │  55,759元  │ 55,759元 │    0%    │
      ├──┼──────┼─────┼─────┤
      │ 92 │  47,951元  │ 47,951元 │    0%    │
      ├──┼──────┼─────┼─────┤
      │ 93 │  47,951元  │ 47,951元 │    0%    │
      └──┴──────┴─────┴─────┘
      被告不顧公告地價與房屋稅下調，造成核定金額卻大幅高
      於87年至90年間各年金額之反常現象，人民權益嚴重受損
      ；訴願決定未予糾正，亦有不當。
  (二)被告主張之理由：
    1.按「個人之綜合所得總額，以其全年左列各類所得合併計
      算之……第五類︰租賃所得及權利金所得……四、將財產
      借與他人使用，除經查明確係無償且非供營業或執行業務
      者使用外，應參照當地一般租金情況，計算租賃收入，繳
      納所得稅。……」「本法第14條第1 項第5 類第4 款所稱
      他人，係指本人、配偶及直系親屬以外之個人或法人而言
      。」「將財產無償借與他人使用者，應由雙方當事人訂立
      無償借用契約，經雙方當事人以外2 人證明確係無償借用
      及法院公證。」為行為時所得稅法第14條第1 項第5 類第
      4 款及同法施行細則第16條第2 項及第88條所明定。
    2.本件原告93年度綜合所得稅結算申報，經中區國稅局台中
      市分局查得其本人持分1/2 之系爭房屋，有星喬行營利事
      業設籍營業，乃核定原告當年度租賃所得47,951元，通報
      被告併課核定當年度綜合所得稅。
    3.原告主張該房屋係與其弟林育樹共有，無償借用林育樹開
      立星喬行，其本人並未收取租金，應免設算租金收入等情
      。經查系爭房屋當年度有星喬行設籍營業，此有卷附營利
      事業登記證可稽，亦為原告所不爭，是系爭房屋既供營業
      使用，參諸首揭規定，即應參照當地一般租金情況，計算
      租賃收入，原告復主張系爭房屋與其弟林育樹共有，其弟
      自有使用權，不須向其承租，又其未曾自該房屋收取任何
      租金，即無所得者無須繳納所得稅等情。按民法第818 條
      規定「各共有人，按其應有部分，對於共有物之全部，有
      使用收益之權。」，惟不得解為「各共有人，對於共有物
      之全部，有使用收益之權。」，是林育樹本於其所有權作
      用，僅得在其應有部分（即二分之一）範圍內，就系爭建
      物之全部行使其使用收益權利。其餘應有部分之權利，係
      屬於原告所有，原告本於其所有權作用，亦在其應有部分
      範圍內，就系爭建物之全部，享有使用收益之權利（如：
      自用、或有償出租、或無償出借等）。再者，所得稅法第
      14條第1項第5類第4款亦規定，將財產借予他人使用須無
      償使用且非供營業或執行業務者使用者，方得以免課租賃
      所得，而本件原告將系爭房屋無償借與其弟為營業使用之
      事實，亦為其所不爭，縱未收取租金，揆諸首揭規定，仍
      應課徵租賃所得，原告所稱，顯有誤解。
    4.至原告主張以前年度申請復查時皆獲核減租賃所得，本年
      度復查決定未予核減乙節，參照92年10月份最高行政法院
      庭長法官聯席會議決議，房屋僅供住宅用者，始受土地法
      第97條第1 項規定之限制，系爭房屋當年度有星喬行設籍
      營業，乃不爭之事實，中區國稅局台中市分局參照當地一
      般租金情況，核定系爭房屋租賃所得，通報被告併課核定
      原告當年度綜合所得稅，並無不合，併予陳明。
      理  由
一、原告起訴主張：系爭房屋係原告與其弟林育樹共有，應有部
    分各1/2，因林育樹開設星喬行，原告乃無償借由林育樹使
    用，詎被告竟設算租金，且不顧系爭房屋所位處之台中市中
    區商圈，商機日趨沒落，公告地價與房屋稅均陸續下調，鄰
    近店家實際已不具出租價值之情事，多有違法，為此訴請如
    聲明所示云云。
二、被告則以：被告予以設算租金，乃係依所得稅法第14條第1
    項第5類第4款之規定，無論原告有無實際收取租金，再系爭
    房屋租賃所得，乃係中區國稅局台中市分局參照當地一般租
    金情況核定通報被告併課核定，於法並無不合等語置辯，求
    為判決駁回原告之訴。
三、本件兩造不爭系爭房屋為原告與其弟林育樹共有，應有部分
    各1/2，系爭房屋，星喬行設立於系爭房屋並有營業登記，
    此有台中市政府營利事業登記證及建築改良物所有權狀在原
    處分卷可稽，堪認為真實。
四、是本件之爭執，在於被告設算核定系爭房屋之租金，於法是
    否有據？
五、經查：
  (一)按行為時所得稅法第14條第1項第5類第4款規定：「個人之
    綜合所得總額，以其全年左列各類所得合併計算之……第五
    類︰租賃所得及權利金所得……四、將財產借與他人使用，
    除經查明確係無償且非供營業或執行業務者使用外，應參照
    當地一般租金情況，計算租賃收入，繳納所得稅。……」
  (二)按量能課稅原則固係稅捐法制的最高理想，惟如將此一理想
    「絕對化」，鑑於稅捐案件具有大量行政之特質，而與稅基
    計算有關之證據資料大部分又掌握在納稅義務人手中，不僅
    稽徵機關之規模勢必擴大到不合經濟效益之地步，整個稅制
    中也將存有太多之漏稅黑洞。結果反而倒過來威脅到量能課
    稅原則之基礎平等原則。因此稅捐法制上稽徵經濟原則之重
    要性也難以忽視。上開所得稅法第14條第1項第5類第4款前
    段之規定，乃是基於稽徵經濟原則，先針對個人就自己之財
    產，通常可以使用收益，不會無償借與他人使用，因此原則
    上以法律擬制之手段視為「其有租金所得產生」，而設算租
    賃所得；但是同時也在該款後段，給予納稅義務人證明「實
    際上確屬無償借用，而沒有取得租金」事實之機會，以緩和
    稽徵經濟原則對量能課稅原則所帶來的衝擊。又即使納稅義
    務人證明了無償借用的事實，但是若其借用之對象為從事營
    利活動的營業人或執行業務人，此時鑑於營利者本身即具有
    營利目標，而無償借用財產予營利者，很難從人情、道義之
    角度予以理解，極有可能涉及不同納稅主體間刻意非常態之
    稅基調整（所得稅法第43條之1 參照），而稅捐稽徵機關進
    行實質調查時，其成本過高，因此又重新強調稽徵經濟原則
    之規範目標，而排除絕對化之量能課稅原則之適用。
  (三)查系爭房屋既供星喬行營業使用，則依上開所得稅法第14條
    第1 項第5 類第4 款之規定，即應按當地一般租金情況，計
    算租賃收入繳納所得稅。
  (四)至原告以民法第818條規定：「各共有人，按其應有部分，
    對於共有物之全部，有使用收益之權。」為據，主張共有人
    之使用，不應設算利息租金收入一節。查上開民法規定僅在
    規範各共有人間本諸物權而生之對外法律關係，但並無一併
    規範各共有人間之內部法律關係，有關各共有人間對共有物
    之使用收益權限，仍須依其應有部分來做分割，各共有人對
    共有物如有超過其應有部分範圍之情形，在私上仍應被評價
    為無償借用，而在稅法上如供營業使用，仍應評價為應按設
    算之租金收入繳稅。
  (五)又查，系爭房屋坐落台中市市○路58號，經中區國稅局查明
    同路63、91、120 及152 號房屋之租金分別為2,010 元、70
    5 元、1,048 元及733 元，得其平均數920 元，乘以系爭房
    屋之營業面積20.03 ，再乘以1/2 （原告應有部分），計算
    原告之租金收入為84,126元，再依成本費用43% 之標準，核
    定原告之租賃所得為47,951元，此有綜合所得稅鄰近租金查
    報單及93年度非自住房屋使用及租賃情形核定表在原處分卷
    可按，於法均無違誤。
  (六)至原告主張以往年度均再依土地法第97條第1 項之規定核減
    云云，查稅捐稽徵機關所定之當地一般租金標準，如所出租
    之房屋，供住宅用者，應受土地法第97條第1 項所設最高額
    之限制；如房屋供業務用（非具營利性）或營業用，則不受
    限制，仍得按所定標準核算租金收入。（最高行政法院92年
    度10月份庭長法官聯席會議決議意旨參照）。是被告就系爭
    年度之租金收入，不再依土地法第97條第1 項之規定核減，
    其適用法律始謂之正確。
六、綜上所述，原告之主張，均非可採，原處分認事用法，並無
    違誤，訴願決定予以維持，亦無不合，原告仍執前詞訴請撤
    銷，為無理由，應予駁回。又本件屬應適用簡易訴訟事件，
    爰不經言詞辯論逕為判決，併予敘明。
據上論結，本件原告之訴為無理由，爰依行政訴訟法第98條第3
項前段，判決如主文。
